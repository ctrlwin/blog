# clipboard 剪切板属性

# navigator.clipboard

navigator.clipboard属性用于访问系统剪切板（不兼容IE）。

js另一个属性document.execCommand()也可以访问系统剪切板，不过即将被废弃。

部分浏览器在使用Clipboard API时，会弹出用户授权弹窗，如果没有授权则不允许读取或修改剪切板内容。
