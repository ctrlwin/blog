# flex 布局，让某个子元素靠右对齐

当父元素设置了`display:flex`时，如果想让它的某个子元素靠右对齐或是靠左对齐，可以这么实现：

## 第一种方法

靠右对齐：

```css
margin-left: auto;
```

靠左对齐：

```css
margin-right: auto
```

## 第二种方法

```css
flex: 1;
text-align: right;// 靠右对齐，left即靠左对齐
```
