## v-on监听多个方法

v-on在`2.4.0+`版本可以使用对象语法实现监听多个方法：

```vue
<div v-on="{click:function1, mouseenter:function2, mouseleave:function3}">text...</div>
```

也可以一个事件绑定多个方法，用 "," 或 ";" 分隔，按顺序执行：

```vue
<div @click="function1(), function2(), function3()">text...</div>
```
